# natural-disasters-NASA
You can see list natural disasters in NASA datadase.
<h2>How to run</h2>
You need Laravel and Vue.

### Run backend:
  ```cd natural-disasters-backend```<br>
  ```php artisan command:init``` <br>
  ```php artisan serve```<br>

### Run frontend:
```cd ../natural-disasters```<br>
```npm i```<br>
```npm run serve```
